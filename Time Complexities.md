## 1- Binary Search tree
- Search = O(log n)
- Insert = O(log n)
- Delete = O(log n)

## 2- Array
### Unsorted
- Search = O(n)
- Insert = O(1)
- Delete = O(n)

### Sorted
- Search = O(log n)
- Insert = O(n)
- Delete = O(n)

## 3- Linked List
- Search = O(n)
- Insert = O(1)
- Delete = O(n)

## 4- Selection Sort
- O(n^2)

## 5- Bubble Sort
- Best case = O(n)
- Worst case = O(n ^ 2)

## 6- Insertion Sort
- Best case = O(n)
- Worst case = O(n ^ 2)

## 7- Merge Sort
- O(n log n)

## 8- Quick Sort
- Best case = O(n log n)
- Worst case = O(n ^ 2)
